# TypingStatisticsAnytime
時間帯別タイピング情報の統計を取って日常におけるタイピングの癖を診断するソフトです

オフラインでのデータ収集を行うことによってセキュリティ上の問題を解消し、加えて動作の軽量化ともなりました。

## データフロー図

graph LR
    subgraph UserPC [ユーザーのPC (オフライン)]
        A[常駐アプリ] -->|統計処理| B(stats.json)
        B -->|ユーザーが中身を確認| B
    end

    subgraph Web [診断サイト]
        C[ブラウザ画面]
        D[サーバー/API]
        E[AI (LLM)]
    end

    B -->|ユーザーが手動アップロード| C
    C -->|データ送信| D
    D -->|統計データをプロンプトに変換| E
    E -->|診断結果・アドバイス| D
    D -->|レポート表示| C
